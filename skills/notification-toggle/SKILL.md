# Notification Toggle

Toggle Claude Code sound notifications on/off.

## Usage

```
/notification-toggle
```

## What it does

Runs the toggle script to enable/disable sound notifications.

## Instruction

Execute this command:
```bash
powershell -ExecutionPolicy Bypass -File "C:\Users\honey\.claude\hooks\notification-sounds\toggle.ps1"
```

Then report the new state (ON or OFF) to the user.
